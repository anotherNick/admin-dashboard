# Admin Dashboard for The Odin Projects

This project was to build an admin dashboard based loosely on the design image provided [here](https://cdn.statically.io/gh/TheOdinProject/curriculum/43cc6ab69fdfbef40d431a65677d2144668930ac/intermediate_html_css/grid/project_admin_dashboard/imgs/dashboard-project.png).

This page was built to be semi-responsive using CSS Grid and Flexbox functionality. The layout expands to fill the viewport and has a minimum width of 1140px. 

It further takes advantage of intermediate CSS tools like the clamp function for scaling fonts, box shadows, opacity, filters, and variables.

Thematically, this design accompanies the [Web Form Project](https://anothernick.github.io/web-form/) as an internal jobs portal for our imaginary megacorp.
